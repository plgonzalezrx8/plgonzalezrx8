## What I'm Building

- **[bluewebcrawler](https://github.com/plgonzalezrx8/bluewebcrawler)** - A Node + TypeScript crawler that maps a site and exports AI-ready markdown.
- **[InfiniMind](https://github.com/plgonzalezrx8/InfiniMind)** - A Docker-first memory sidecar for OpenClaw with a dedicated memory service and bridge plugin.
- **[OpenClaw-Skills](https://github.com/plgonzalezrx8/OpenClaw-Skills)** - Custom skills that extend OpenClaw with specialized commands and integrations.
- **[CredForge](https://github.com/plgonzalezrx8/CredForge)** - A credential processing and analysis toolkit for NTDS dumps, password files, and credential lists.
- **[Next-Gen-Rubber-Ducky-Ninja](https://github.com/plgonzalezrx8/Next-Gen-Rubber-Ducky-Ninja)** - A modern WinUI 3 and .NET 8 toolkit for testing, validating, and encoding DuckyScript payloads.
